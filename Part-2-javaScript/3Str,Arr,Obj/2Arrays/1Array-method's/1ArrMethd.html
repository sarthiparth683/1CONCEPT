Array-Method's

  1. .push() - add elements to the end
  2. .pop() - removes the last element
  3. shift() removes the first element from the array
  4. unshift() adds to the beginning.
  5. concat() merge arrays
  6. slice() extract portions
  7. indexOf() finds elements
  8. includes() checks for existence
  9. join() creates strings
  10. reverse() reverses the array
  11. splice() modifies arrays by removing
