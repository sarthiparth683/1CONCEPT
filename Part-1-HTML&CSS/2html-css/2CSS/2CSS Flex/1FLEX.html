Flex-Parent Properties:-

  1. display:flex
  2. flex-direction:- Row,Column, rowreverse, and column-reverse, (gives Direction rowWise or columnWise).
  3. justify-content:- Center,space-between,space-around,space-evenly,flex-start,flex-end,(Everything Row-wise, Horizontally).
  4. align-items:- Center,flex-start | flex-end | | baseline | stretch;(Everything, Column-wise, Vertically)
  5. flex-wrap:- wrap,nowrap,wrap-revese,(it wraps the items)

    Flex-Child Properties:-

  1. .box1 div:nth-child(1)
  2. align-self:- overrides alignment set on parent
  3. flex-grow:- (Defaults to 0)
  4. flex-shrink:- (Defaults to 1)
  5. order
  6. flex-basis
