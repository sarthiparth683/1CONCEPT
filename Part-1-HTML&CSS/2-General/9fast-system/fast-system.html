In addition to Prefetch, Temp, and %Temp%, there are other folders and system utilities you can access via the Run command in Windows to delete unnecessary files and potentially improve system performance: 1. Recent Files Run Command: recent This opens a folder containing shortcuts to files you've recently accessed. You can delete these shortcuts (not the actual files) to clear recent history.
  14. 2. Recycle Bin Run Command: shell:RecycleBinFolder This opens the Recycle Bin. Empty it to free up disk space.
  15. 3. Temporary Internet Files (Internet Cache) Run Command: inetcpl.cpl This opens the Internet Properties window. Under the General tab, click Delete in the Browsing history section to clear your temporary internet files, cookies, and history.
  16. 4. DNS Cache Run Command: cmd (Opens Command Prompt) In the Command Prompt, type ipconfig /flushdns and press Enter. This will clear the DNS cache, which can help with connectivity issues and speed up DNS resolution.
  17. 5. System Restore Points Run Command: sysdm.cpl (Opens System Properties) Go to the System Protection tab, click Configure on your system drive, then click Delete to remove all system restore points except the most recent one.
  18. 6. DirectX Shader Cache Run Command: cleanmgr (Opens Disk Cleanup) Select your system drive (C:), and check the DirectX Shader Cache option. This will remove old DirectX cache files.
  19. 7. Downloaded Program Files Run Command: cleanmgr In the Disk Cleanup window, select Downloaded Program Files (ActiveX controls and Java applets that are cached from the internet).
  20. 8. Old Windows Update Files Run Command: cleanmgr In Disk Cleanup, select the Windows Update Cleanup option. This deletes old Windows Update files that are no longer needed.
  21. 9. Windows Error Reporting Files Run Command: cleanmgr In Disk Cleanup, select System queued Windows Error Reporting to delete old error reports.
  22. 10. Hibernate File Run Command: cmd (Opens Command Prompt) Type powercfg -h off and press Enter. This will disable hibernation and delete the hiberfil.sys file, which can take up a significant amount of disk space.
  23. 11. Prefetch Advanced Option Run Command: regedit (Opens Registry Editor) You can tweak the Prefetch settings in the registry. Navigate to: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters Modify the EnablePrefetcher value: 0: Disable Prefetch 1: Prefetch only for application startup 2: Prefetch only for boot 3: Prefetch for both application and boot (default)
  24. 12. Driver Store Cleanup Run Command: cleanmgr Check Device driver packages in Disk Cleanup to remove unused driver packages. These methods, when used carefully, can help improve your system's performance. Be sure not to delete any essential system files, and always double-check what you are removing, especially when accessing advanced features like the Registry Editor.
