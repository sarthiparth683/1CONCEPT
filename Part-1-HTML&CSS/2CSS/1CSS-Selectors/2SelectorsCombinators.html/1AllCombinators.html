4 Types of Combinators in CSS

  • Descendant selector - (space):-it will select all the tags , nested one also.
  • Child selector - (>) :-selects all child inside it but not nested ones.
  • Sibling selector - (~):- Select all his neighbour brother BUT not himself.
  • Adjacent sibling selector - (+) :- same as sibling(~) but it selects only one.
