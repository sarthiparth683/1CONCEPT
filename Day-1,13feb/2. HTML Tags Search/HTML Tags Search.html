Tag names

  1. b tag

    The "b" tag in HTML is used to format text as bold.

  2. i tag

    The "i" tag mainly serves the purpose of displaying text in italics.

  3. u tag

    The "u" tag in HTML is used for unarticulated annotations, not just underlining text. This means it indicates that the enclosed text has some special meaning, but the nature of that meaning isn't directly conveyed by the tag itself.

  4. select tag

    The "select" tag in HTML is used to create a drop-down list, also known as a select element. This allows users to choose one or more options from a predefined set. It's commonly used within forms to collect user input.

    The "hr" tag in HTML stands for horizontal rule and is used to visually separate content on a webpage. While it often appears as a simple horizontal line, its primary purpose is to represent a thematic break.

  7. br tag

    The "br" tag in HTML is used to create a line break within text. It's an empty tag, meaning it doesn't have a closing tag like most other HTML elements.
